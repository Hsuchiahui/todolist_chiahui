# todolist_chiahui
## Basic Feature [(app.py)](https://github.com/Hsuchiahui/todolist_chiahui/blob/master/app.py)
☆ 網頁介面提供以下功能：
1. 新增任務 【C】
2. 列出任務 【R】
3. 修改任務 【U】
4. 刪除任務 【D】

## UI/UX
1. 即使不透過使用教學，也能自然的完成任務：
   網頁分為輸入框區域與顯示區域。讓使用者可以直覺地輸入代辦事項與日期。
2. 易於找到所有關鍵的資料，降低認知負擔：
    網頁的顯示區域位於提交按鈕下方。顯示區的左半部顯示代辦事項，右半部顯示日期。
